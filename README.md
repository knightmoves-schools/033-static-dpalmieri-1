the Account.cs file
- should create a `public static` `class-level` string variable named `Name`
- should create a `public static` method named `ChangeName` that takes a new name, sets `Name` with the new value, and returns the new name
- should create a `static` constructor that sets the `Name` variable to "Main Account"
